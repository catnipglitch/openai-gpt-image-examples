# openai-gpt-image-examples

OpenAI の gpt-image-1 / gpt-image-1.5 モデルを使った画像生成のサンプル集です。

## 構成

- `image_1_cli/` — gpt-image-1 を使った CLI ツール（[README](image_1_cli/README.md)）
- `examples_openai/` — gpt-image-1.5 のサンプルスクリプト・ノートブック
- `examples_colab/` — Google Colab 用ノートブック
- `docs/` — API リファレンス・仕様書などのドキュメント

## セットアップ

```bash
# 依存パッケージのインストール（uv を使用）
uv sync

# OpenAI API キーを設定
export OPENAI_API_KEY="your-api-key"
```
